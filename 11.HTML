What is HTML?

HTML stands for HyperText Markup Language. HTML, which stands for Hypertext Markup Language, is the standard language used to create and design web pages. It's a markup language, not a programming language, and it's used for structuring content on the web. HTML uses a system of tags and attributes to format text, multimedia, and other elements on a web page

What is Semantic HTML?

A semantic element in HTML is a tag that carries meaning about the structure and content of a web page. It provides information about the type of content contained within the tags and its role on the web page. Semantic elements help both web browsers and developers understand the purpose and significance of different parts of a webpage.

Common Semantic HTML Tags

Here are some common semantic HTML tags:

  • <aside>: Defines content that is not directly related to the main content of a document.
  • <section>: Defines a section of a document.
  • <nav>: Defines a set of navigation links.
  • <header>: Defines a header for a document or section.
  • <article>: Defines an article or self-contained piece of content.
  • <footer>: Defines a footer for a document or section.
  • <main>: Defines the main content of a document.
